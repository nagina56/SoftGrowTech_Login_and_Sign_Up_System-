# SoftGrowTech_Login_and_Sign_Up_System

A secure CLI-based Login &amp; Signup System built with Python features  password hashing, salt encryption, JSON storage, session management, and input validation.

## 🚀 Features
- User Registration (Sign Up)
- User Authentication (Login)
- Secure Password Hashing (SHA-256)
- JSON File Storage
- Duplicate Username Validation
- Password Confirmation Check
- Error Handling
- Command Line Interface (CLI)

## 🛠️ Technologies Used
Python 3, JSON File Handling, Hashlib (SHA-256), Command Line Interface (CLI)

## ▶️ How to Run
1. Clone the repository
2. Open terminal in the project folder
3. Run the command:

python main.py
